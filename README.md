# 🏦 Simple C++ Bank System

## 📌 Description
A simple console-based banking system built using C++.
The program allows users to:
- Create an account
- Set a 4-digit PIN
- Deposit money
- Withdraw money
- View account details

---

## 🚀 Features
- 🔐 PIN verification for secure access
- 💰 Deposit money into account
- 💸 Withdraw money with balance check
- 📄 Display account information
- 🔁 Continuous menu (loop system)

---

## 🛠️ Technologies Used
- C++
- Standard Library (<iostream>, <string>)

## 🧠 How It Works
1. User enters their name and sets a 4-digit PIN.
2. The system displays a menu:
   - Deposit
   - Withdraw
   - Display account
   - Exit
3. Each operation requires PIN verification.
4. The program runs in a loop until the user exits.

---

## ⚠️ Notes
- The PIN must be exactly 4 digits.
- No limit on PIN attempts (can be improved).
- Data is not saved after program exits (no file handling yet).
